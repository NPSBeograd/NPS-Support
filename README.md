## 🐛 Report an Issue

Found a bug or have a suggestion? We'd love to hear from you!

### **Quick Start:**

1. **[Create New Issue](https://github.com/NPSBeograd/NPS-Support/issues/new/choose)** - Choose from predefined templates
2. **[View All Issues](https://github.com/NPSBeograd/NPS-Support/issues)** - Browse existing issues
3. **[Project Board](https://github.com/orgs/NPSBeograd/projects/19)** - Track progress and status

### **Issue Templates:**

- 🐛 **Bug Report** - Report a problem or unexpected behavior
- ✨ **Feature Request** - Suggest a new feature or enhancement  
- 📝 **Documentation** - Request documentation improvements
- ❓ **Question** - Ask for help or clarification

### **Issue Guidelines:**

✅ **Before submitting:**
- Search existing issues to avoid duplicates
- Check the [documentation](https://npsbeograd.github.io) first
- Gather relevant information (BC version, extension version, error messages)

✅ **When submitting:**
- Use descriptive titles
- Provide step-by-step reproduction steps
- Include screenshots or error logs if applicable
- Specify your Business Central version and environment
